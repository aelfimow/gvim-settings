# gvim-settings
gVim related settings etc.

## vimrc

```
set nocompatible
set hlsearch
set noswapfile
set nobackup
set number
set guifont=Lucida\ Console:h16
set tabstop=4
set shiftwidth=4
set expandtab
set cindent
set incsearch
syntax on
set nowrap
```
## utf-8 settings
```
set encoding=utf-8
set fileencoding=utf-8
```
